### Hobby RPN 電卓用ケース

![DSC_3754](https://github.com/user-attachments/assets/b7f547db-85ae-4ef0-b735-2ea0bec7a0be)

Blender でデザインしました。上のケースは Bambu Lab P1S の標準ノズル（0.4mm）でプリントしたものです。

- 平面部（前面、底面）がビルドプレートに接するような姿勢で出力してください．
- 多色プリントができる3Dプリンタ（AMS付きの Bambu Lab プリンタなど）では、main.stl と colored.stl を１つの部品として読み込んでください（[BambuStudioでの操作手順例](https://www.youtube.com/watch?v=ASEnFGKUiak)）。
- 多色プリントが出来ない3Dプリンタでは、mono.stlをプリントしてください。穴が空きますが、ある程度読めると思います。
- 底蓋の bottom.stl は共通です。

裏蓋の固定のために 2mm x 8mm のタッピングスクリューを４本使用します．

<img width="500" src="https://user-images.githubusercontent.com/86639425/126858054-ffea1137-84ab-4712-97ba-c734e60ee0c0.jpg">

かなり薄手に設計したので（主な肉厚は1mm）厚みがオリジナルより 2mm ほど薄くなります。ただし、側面が覆われているため上下・左右に3mmほど大きいです。
